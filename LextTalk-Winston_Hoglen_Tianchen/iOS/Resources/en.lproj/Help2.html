    Share
    Share the map you are seeing with others via Twitter, Facebook or email. Tap on it and select how you want to share the map. You have to login to the sharing service the first time you use the service. After that, you can share directly.

    Locate
    Use this button to easely find yourself in the map.

    Users on the map
    This button will bring you to a list of the users that are currently in the map area you are seeing. Use the search bar (scrolling up the table) to search within the list. You can search by user name, learning language or native language.

    Search
    Use this button to search for users. You can search in the current area of the map, or in the whole world. You can search by user name, learning or native language. Use this feature to find users to chat in the languages you are learning!

    Help
    Brings you here.

  • Chat

    This view shows you a list of users you have chats with. Tap on any entry to go on chatting with that user. When you are chatting, you can tap the translation buttons which are shown next to the other user's messages. You can also chat with multiple users at the same time by using chatrooms. You can create your own chatrooms by providing a name and a language. Other users can see and join them!

  • Chatrooms

    See who of your contacts are already in Lext Talk!

  • Profile

    Use this tab to sign in or sign up, it is free and you don't have to provide any personal information, just a login name and password. When you sign in, you can change your status message and the languages you are learning and speaking. You can also select if your user will be shown in the map in your real location or in another position..

  • Translator

    Use this tab to translate texts from any supported language to another. By clicking the red button "To Chat" you can copy the translated text to the chat window. You can also add any transalation to your personal dictionaries. Use the top right button to access those dictionaries
